# 대기업 자소서 샘플 데이터 100건 생성 및 삽입 계획

형의 요청대로 대기업 지원자들의 리얼한 자소서 데이터를 100건 생성해서 DB에 채워둘게. 각 에세이 문항별로 500자 이상의 충분한 분량을 확보해서 RAG 성능을 제대로 테스트할 수 있게 만들 거야.

## Proposed Changes

### [Script] 데이터 생성 스크립트 작성

#### [NEW] [generate_sample_data.py](file:///home/foenix/workspace/spring-rag/src/main/resources/generate_sample_data.py)
- 파이썬을 이용해 100명의 가상 지원자 정보를 생성합니다.
- 성함, 수험번호(2026_001~100)를 생성합니다.
- 대기업 자소서의 단골 문항(성장과정, 장단점, 직무역량, 지원동기) 4가지를 테마로 하여 각각 500자 내외의 텍스트를 생성합니다.
- 생성된 데이터를 `INSERT` 문으로 만들어 SQL 파일로 출력하거나 직접 DB에 주입합니다.

### [Database] 데이터 삽입

- 생성된 100건의 데이터를 `test.rec_appli_mas` 테이블에 일괄 삽입합니다.
- 기존에 들어있던 테스트용 '김철수' 데이터와 충돌나지 않도록 처리합니다.

## 데이터 구성 구성 (예시)
- **문항 1**: 성장과정 및 가치관 (도전 정신, 성실함 등)
- **문항 2**: 본인의 장단점 및 보완 노력
- **문항 3**: 직무 관련 핵심 역량 및 프로젝트 경험
- **문항 4**: 지원 동기 및 입사 후 포부
